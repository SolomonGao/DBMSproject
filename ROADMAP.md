### Phase 1: 产品定位重塑

- [x] 撰写产品愿景文档
- [x] 设计新架构图

### Phase 2: 数据架构优化
- [x] 创建7个预计算表（daily_summary, region_daily_stats, geo_heatmap_grid等）
- [x] 编写ETL Pipeline脚本（每日2AM自动运行）
- [x] 表分区改造（按日期分区）
- [x] 2024年全年数据批处理（366天，1589万事件）
- [x] 生成32.8万事件指纹

### Phase 3: 意图驱动工具重构
- [x] 开发6个意图驱动工具（search_events, get_event_detail等）
- [x] 实现事件指纹系统（标准指纹+临时指纹）
- [x] Router集成（Qwen 2.5B本地意图识别）
- [x] CLI提示词同步更新

### Phase 3.5: 技术融合增强（txx_docker 合并）
- [x] **RAG语义搜索**: ChromaDB + all-MiniLM-L6-v2 向量数据库
- [x] **SQL查询缓存**: LRU + TTL 双机制缓存
- [x] **并行查询优化**: 5查询并发执行（get_dashboard）
- [x] **流式查询**: 大数据量分块读取，内存友好
- [x] **空间-时间叙事**: 多跳推理、因果链分析
- [x] 工具集整合优化（15个工具）

## 🎯 下一步行动建议

### 立即可以开始（高优先级）

1. **前端框架选型**
   ```
   推荐: React 18 + TypeScript + Vite + Ant Design
   理由: 生态成熟、类型安全、开发体验好
   ```

2. **地图组件选型**
